# OpenIdea · 共笔 （🚧施工中）

> 开局由我，后续由你；每一章都由大家投票决定。

**共笔** 是一个众创协作小说平台。开发者只负责埋下故事的“引线”，后续的一切由社区共同完成。

我们以**季度**为周期：每月一轮续写，全民投票定胜负。票数第一成为**正线**，第二成为 **if 线**——两条平行世界，同步演进。

---

## 📖 季度玩法流程

### 🎯 整体结构

| 阶段 | 时间 | 核心事件 | 谁来决定 |
| :--- | :--- | :--- | :--- |
| **开局月** | 第1个月 | 开发者发布开局 → 用户续写第1章 → 全民投票 | ✅ 用户投票 |
| **演进月①** | 第2个月 | 双线分别续写第2章 → 全民投票 | ✅ 用户投票 |
| **演进月②** | 第3个月 | 双线分别续写第3章 → 全民投票 | ✅ 用户投票 |
| **季末** | 3个月末 | 季度成书 + 下季预告 | - |

---

### 📅 详细流程

#### 【第1个月：开局月】

**第1周：开局发布**
- 开发者发布小说**开头**（3000-5000字）
- 明确世界观、核心悬念、主要人物
- 开放**第1章续写投稿通道**

**第2-3周：自由续写**
- 用户基于开头续写**第1章**（2000-3000字）
- 每人限投1篇
- **所有投稿自动进入投票池**（无人工筛选）

**第4周：全民投票定双线**
- 公示所有投稿作品
- 每位注册用户**1票**
- **票选结果**：
  - **第1名** → 正线·第1章
  - **第2名** → if线·第1章
- 结果公布后，开放第2章续写预告

---

#### 【第2个月：演进月①】

**此时已有两条平行故事线：**
- **正线 A**（上月投票第1名）
- **if线 B**（上月投票第2名）

**第1周：双线起点发布**
- 发布**正线A·第1章** + **if线B·第1章**全文
- 开放**第2章续写通道**（投稿时需选择续写哪条线）

**第2-3周：双线并行续写**
- 用户分别针对正线A和if线B续写**第2章**
- 每条线每人限投1篇
- **所有投稿自动进入对应线路投票池**

**第4周：双线并行全民投票**
- 公示正线A所有投稿 + if线B所有投稿
- **投票规则**：每人每条线可投1票（即每人共2票）
- **票选结果**：
  - 正线A得票第1名 → 正线·第2章
  - if线B得票第1名 → if线·第2章

---

#### 【第3个月：演进月②】

流程与第2个月完全一致：

- **第1周**：发布第2章双线全文，开放第3章投稿
- **第2-3周**：双线并行续写
- **第4周**：双线并行全民投票
- **票选结果**：
  - 正线A得票第1名 → 正线·第3章
  - if线B得票第1名 → if线·第3章

---

#### 【季末：收官与传承】

**第3个月末**

- **成果发布**：整理并发布季度完整正线合集 + if线合集（标注每章作者）
- **作者荣誉**：公示所有入围作者、当选作者，颁发虚拟勋章
- **下季预告**：发布下一季度新开局（全新世界观 / 高人气if线转正）

---

### 📊 核心机制

| 机制 | 说明 |
| :--- | :--- |
| **投票权** | 全民投票，**无人工干预**，票数高低定胜负 |
| **入围机制** | **所有投稿自动进入投票池**，仅人工筛选是否符合公序良俗|
| **每人票数** | 开局月：每人1票；演进月：每人每线1票（共2票） |
| **投稿限制** | 每人每轮每线限投1篇 |
| **字数建议** | 开局 3000-5000 字，续写 2000-3000 字 |
| **if线价值** | 平行世界延续，未来可投票决定是否“转正” |
